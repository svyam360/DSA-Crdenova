# DSA Lending Software

Enterprise-grade DSA (Direct Selling Agent) Loan Management Platform

## Technology Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Backend**: Spring Boot (Java 17)
- **Architecture**: Svayam360 Pattern

## Project Structure

```
src/
├── features/          # Feature modules
├── shared/            # Shared components
├── services/          # API services
├── config/            # Configuration
├── constants/         # Constants
├── types/             # TypeScript types
└── routes/            # Routing
```

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Features

1. Master & Configuration
2. DSA & User Onboarding
3. Lead Management
4. Eligibility & Pre-Screening
5. Loan Application
6. Document & KYC Management
7. Credit / Underwriting
8. Approval & Disbursement
9. Commission & Payout
10. Reports & Dashboards
11. Admin & User Access Management

## Environment Variables

- `VITE_USE_DEMO_DATA`: Enable/disable demo data mode
- `VITE_API_BASE_URL`: Backend API URL
- `VITE_APP_NAME`: Application name
